Posts API

Update a post

PATCH api/v1/posts/:id

Parameters

Name Description
data[id] The id of post to update
data[type] The type of resource to update
data[attributes][published-at] The publish date of the post in iso8601 format
data[attributes][banner-url] The banner of the post
data[attributes][title] The post title
data[attributes][content] The post content
data[attributes][tags] The post tags
data[attributes][kind] The post kind
data[relationships][group][data][type] The group type
data[relationships][group][data][id] The associated group id
