Sermons API

Update a sermon

PATCH api/v1/sermons/:id

Parameters

Name Description
data[id] The id of the sermon to update
data[type] The type of resource to update
data[attributes][published-at] The publish date of the sermon is iso8601 format
data[attributes][banner-url] The banner of the sermon
data[attributes][audio-url] The audio file of the sermon
data[attributes][name] The sermon name
data[attributes][notes] The sermon notes
data[attributes][speaker] The sermon speaker
data[attributes][series] The sermon series
data[attributes][tags] The sermon tags
data[relationships][group][data][type] The group type
data[relationships][group][data][id] The associated group id
